### 🛠 Repository Guidelines
To maintain a structured archive, we follow these naming conventions:

*   **Theses:** `thesis-<topic>-<year>-<month>`  
    *Final academic records including text, code, and defense materials.*
*   **Projects:** `<project_name>-<module>`  
    *Active R&D, often integrating multiple modular components.*
*   **General Purpose:** `tool-`, `lib-`, or `course-`  
    *Utilities, shared libraries, or educational resources.*
*   **Other:** Descriptive names for specialized repositories (e.g., `awesome-bulgarian-ai`).
